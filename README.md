## What Each Sample Includes

Each folder includes:

* `brief.md` — the project goal, business context, offer, and copy objective
* `target-audience.md` — audience profile, pain points, motivations, and objections
* `final-copy.md` — the finished copy sample
* `before-and-after.md` — a weak version, improved version, and explanation of the changes
